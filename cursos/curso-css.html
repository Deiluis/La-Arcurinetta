Curso de CSS 3

Algo de historia sobre CSS

El origen

En 1994, el W3C o World Wide Web Consortium (la organización encargada de crear los estandares web) propuso un concurso para la creación de un lenguaje web capaz de darle estilos a HTML. Se presentaron en total nueve propuestas, pero solo dos fueron tomadas en cuenta: CHSS o Cascading HTML Style Sheets, creado por Håkon Wium Lie, y SSP o Stream-based Stylesheet Proposal, creado por Bert Bos.

A finales de 1994 e inicios de 1995, Lie y Bos decidieron unirse para definir un nuevo lenguaje que tomaba lo mejor de cada una de estas propuestas. De esa union nacio CSS o Cascading Style Sheets.

¿Cómo siguió?

A principios de 1995 el W3C decide apostar por el desarrollo y estandarización de CSS, añadiendolo a su grupo de trabajo junto con HTML. Para 1996 se publicó la primera recomendación oficial, llamada CSS nivel 1, o CSS 1.

En mayo de 1998 se publica la segunda recomendación de CSS, CSS 2. En ese mismo año comienza el desarrollo de CSS 3, el cual se fue actualizando y se utiliza hasta hoy en día.

Estructura básica de CSS

CSS es un lenguaje que se utiliza para dar estilos a los distintos elementos que creamos en HTML. Para esto, cuenta con una estructura conformada por 4 componentes básicos:

  • El selector
  • Las llaves
  • Las reglas CSS

Selectores

Un selector es un componente que se utiliza mismamente para seleccionar un elemento que tenemos en nuestro HTML y aplicar estilos sobre el mismo. Este puede ser varios tipos, los cuales abordaremos en un próximo capítulo. En este capítulo hablaremos sobre los selectores por elemento los cuales se definen con el mismo nombre del elemento el cual queremos seleccionar.

Llaves

Seguido de cada selector, se colocan unas { } (llaves, o llaves curvas), las cuales encerrarán todas las propiedades que tendrá el elemento seleccionado.

Reglas CSS

Las reglas CSS, establecen o modifican las propiedades o caracteristicas del elemento seleccionado con el selector previamente escrito. Cada una de estas reglas se compone por:

  • El nombre de la propiedad
  • Dos puntos :
  • El valor de la propiedad
  • Un punto y coma ;

El nombre de la propiedad indica que propiedad queremos establecer o modificar. Los dos puntos indican que se establecerá el valor. El valor es el parámetro a establecer o modificar de la propiedad. Finalmente, el punto y coma indica el fin de la regla, y la posibilidad de iniciar una nueva.

Veamos un ejemplo gráfico:

Ejecutar Reiniciar Copiar
h1 {
color: red;
}

Maneras de usar CSS

Existen 3 formas diferentes de aplicar estilos a nuestros elementos HTML con CSS. Estas formas son:

  1. Utilizando el atributo style
  2. Utilizando el elemento style
  3. Vínculando una hoja de estilos externa

Atributo style

En cada elemento HTML se puede aplicar el atributo style, utilizando como valor las reglas CSS que queremos que se apliquen a dicho elemento, quedando de la siguiente manera: <h1 style="color:red">Titulo</h1>.

Aunque esta es la forma menos recomendada, ya que el código HTML se mezcla con el código CSS, y cada propiedad que se aplica a los elementos es individual; no pueden utilizarse en varios elementos a la vez si no se escribe en cada elemento.

Elemento style

En el body de nuestro archivo HTML puede aplicarse el elemento <style></style> utilizando como contenido el código CSS que deseamos aplicar, quedando de la siguiente manera:

<style>
h1 {
color: blue
}
</style>

Utilizar esta forma es más recomendada que utilizar el atributo style ya que el código CSS no esta directamente sobre los elementos, pero aún sigue ligado al código HTML y no puede utilizarse en otros archivos, por lo que no es la forma más recomendada.

Hoja de estilos externa

En el head de nuestro archivo HTML, puede incluirse el elemento link el cual se utiliza para vincular archivos CSS externos, utilizando como atributos rel donde se introduce el tipo de vínculo, y href donde se introduce la ruta de donde esta ubicado nuestro archivo CSS, incluyendo la extensión .css, quedando de la siguiente manera: <link rel="stylesheet" href="style.css">.

Esta es la forma más recomendada, ya que mantiene total independencia del código HTML y puede aplicarse en varios archivos HTML.

Tipos de selectores

Existen diferentes tipos de selectores, veremos los más comunes y como funcionan. Por ejemplo:

  • Por elemento
  • Por clase
  • Por identificador único o id
  • Por jerarquía

Estos son los tipos de selectores más comunes, aunque existen otros que podemos utilizar:

  • Por atributo
  • Por pseudoclase

Selectores por elemento

Los selectores por elemento refieren a selectores generales los cuales se basan en el nombre de los elementos. Todas las propiedades que se apliquen a estos selectores, se le aplicaran a los elementos que le correspondan. Por ejemplo los selectores h1, p, div, o *. Este ultimo, el * es llamado el selector universal, ya que identifica a todos los elementos de HTML.

Veamos un ejemplo gráfico:

Ejecutar Reiniciar Copiar
h1 {
color: blue;
}

Selectores por clase

Los selectores por clase aplican propiedades a elementos los cuales tengan una clase especifica. Por ejemplo, si un elemento o serie de elementos poseen la clase parrafo, para armar el selector se utiliza un . (punto) y el nombre de la clase, quedando el selector como .parrafo.

Veamos un ejemplo gráfico:

Ejecutar Reiniciar Copiar
.parrafo {
color: red;

Selectores por id

Los selectores por id (o identificador) aplican propiedades a un único elemento el cual tenga un id especifico. Por ejemplo, si un elemento posee el id titulo, para armar el selector se utiliza un # (hashtag) y el nombre del id, quedando el selector como #titulo.

Selectores por jerarquía o descendencia

Cuando un elemento está dentro de otro elemento, se dice que el primer elemento es hijo del segundo elemento, y el segundo elemento es padre del primer elemento. A esto se le llama la "descendencia de elementos" y puede utilizado para seleccionar elementos en CSS.

La descendencia de selectores puede armarse con cualquier otro tipo de selectores. Por ejemplo, si queremos seleccionar un b que esta dentro de un p, podemos armarlo de la siguiente manera: p b (notese el espacio entre selectores). También puede armarse con clases: .clase1 .clase2 (en este caso estamos seleccionando un elemento de con la clase .clase2, dentro de otro elemento con la clase clase1).

Esto también funciona con ids, por ejemplo: #elemento1 #elemento2, o de manera mixta, con cualquier otro tipo de selector: #elemento1 .clase2 h2 (en este caso, estamos seleccionando un h2, dentro de un elemento con la clase clase2, dentro de otro elemento con el id elemento1).

Veamos un ejemplo gráfico:

Ejecutar Reiniciar Copiar
p b {
color: orange;
}

Selectores por atributo

Los selectores por atributo utilizan los atributos de los elementos para seleccionar a estos ultimos. Estos selectores pueden armarse de dos maneras:

  • Utilizando el nombre del elemento, y el nombre del atributo entre corchetes. Imaginemos que queremos seleccionar un h6 el cual tiene el atributo grande. El selector se armaría de la siguiente manera: h6[tamano].
  • Utilizando el nombre del elemento, nombre del atributo, y un valor para el atributo, el cual lo vuelve más especifico. El selector se armaría de la siguiente manera: h6[tamano="grande"].

Veamos un ejemplo gráfico:

Ejecutar Reiniciar Copiar
h6[tamano] {
color: blue;
}

h6[tamano="grande"] {
color: red;
}

Selectores por pseudoclase

Las pseudoclases son clases predefinidas en CSS las cuales poseen ciertas funcionalidades que son aplicadas a los elementos seleccionados, o también idenfitican a los elementos de otras maneras. Algunas de las pseudoclases más utilizadas y sus funciones son:

  • :hover: Aplica estilos al pasar el mouse sobre el elemento o tocarlo (en celulares).
  • :active: Aplica estilos sobre el elemento al hacer click sobre él (todo el tiempo que dure el click).
  • :first-child: Si existe más de un elemento al que se esta seleccionando, con esta pesudoclase se selecciona especificamente al primero de todos ellos. Si solo existe un elemento de ese tipo, selecciona ese único.
  • :last-child: Actúa de manera similar a first-child, con la diferencia de que selecciona al último de esos elementos.
  • :nth-child(n): Si existe más de un elemento al que se esta seleccionando, con esta pesudoclase se selecciona especificamente al numero de elemento que se introduzca entre parentesis; es decir, si quisieramos seleccionar al quinto elemento de una serie de elementos con una misma clase, debemos introducir un 5 entre los parentesis, quedando: :nth-child(5).

Para utilizar cualquiera de estas pseudoclases, se utiliza cualquier selector que queramos usar para referenciar al elemento, seguido de : (dos puntos) y el nombre del pseudoelemento. Por ejemplo .clase:hover.

Veamos un ejemplo gráfico:

Ejecutar Reiniciar Copiar
p:hover {
color: violet;
}

p:active {
color: red;
}

p:first-child {
color: blue;
}

p:last-child {
color: green;
}

p:nth-child(3) {
background-color: orange;
}

Especificidad y cascada

En CSS existen dos conceptos que serán cruciales a la hora de escribir nuestro código: La especificidad de selectores y la cascada de estilos.

Especificidad

La especificidad se refiere a que tan especifico es un selector al referenciar un elemento HTML. Mientras mayor especificidad tenga un selector, más relevancia tendrá sobre los otros selectores que se le apliquen al mismo elemento. Existen dos ordenes de especificidad: Según las formas en las que se utiliza CSS, como también dentro del código CSS.

El orden de especificidad según las formas en las que se utiliza CSS es:

  1. Estilos en línea (atributo style)
  2. Elemento style dentro del HTML
  3. Hoja de estilos externa

El orden de especificidad dentro del código CSS es:

  1. Sentencia !important
  2. Identificadores
  3. Clases
  4. Pseudoclases
  5. Atributos
  6. Elementos
  7. Pseudoelementos

Cascada

Como su propio nombre lo indica, CSS es un lenguaje de hojas de estilos en cascada. La cascada refiere a que los selectores y las reglas CSS se escriben una debajo de la otra. Ahora ¿Qué pasaría si escribimos dos reglas con valores diferentes en selectores separados? Pueden pasar dos cosas:

  1. Si ambos selectores tienen la misma especificidad, actúa la cascada y se aplican los estilos del último selector escrito.
  2. Si los selectores tienen una especificidad diferente, se aplicaran los estilos del selector con mayor especificidad, sin importar la cascada.

Veamos un ejemplo gráfico:

Ejecutar Reiniciar Copiar
p:hover {
color: violet;
}

p:active {
color: red;
}

p:first-child {
color: blue;
}

p:last-child {
color: green;
}

p:nth-child(3) {
background-color: orange;
}
