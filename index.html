Introducción

Al crear páginas web, se requiere de tres lenguajes web para su pleno funcionamiento, estos lenguajes son HTML, CSS y JavaScript. HTML se utiliza para estructurar los elementos de la página, CSS les da estilo a estos elementos, tales como colores, margenes o incluso animaciones. Por último, JavaScript le da interactividad con el usuario, pudiendo modificar la página dinámicamente.

En estos cursos se abordarán únicamente HTML para la estructura de las páginas web y CSS para su estilo.

Cursos

HTML

HTML o HyperText Markup Language es un lenguaje de marcado de hipertexto utilizado en desarrollo web para estructurar la composición de los documentos o páginas web.

Ir al curso
<!DOCTYPE html>
<html>
<head>

<title>Curso de HTML</title>

</head>

<body>

<h1>Esto es un título</h1>
<p>Esto es un párrafo</p>

</body>
</html>

CSS

CSS o Cascading Stylesheets es un lenguaje de hojas de estilo utilizado en desarrollo web para dar estilos a los documentos o páginas web.

Ir al curso
h1 {
font-size: 40px;
color: #0070b6;
}

div {
margin: 40px;
padding: 20px;
}

img {
filter: blur(10px);
}

Sobre nosotros

"La Arcurinetta" nació como un proyecto web en la clase de Laboratorio de Aplicaciones, su nombre proviene del profesor de esta materia, Hugo Arcuri. En La Arcurinetta buscamos instruir a los entusiastas a programadores y programadoras para aprender desarrollo web de una forma sencilla y rápida con tutoriales concisos y ejercicios de práctica.
